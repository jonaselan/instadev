# InstaDev

Simple app for study react and nodejs

## Install

### backend

```bash
$ cd backend
# dependencies
$ yarn install
# start server
$ yarn dev
```

### frontend

```bash
$ cd frontend
# dependencies
$ yarn install
# start server
$ yarn start
```
