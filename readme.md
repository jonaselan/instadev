# InstaDev

Simple Instagram clone build with nodejs and react

## Install

### backend

```bash
$ cd backend
# dependencies
$ yarn install
# start server
$ yarn dev
```

### frontend

```bash
$ cd frontend
# dependencies
$ yarn install
# start server
$ yarn start
```
